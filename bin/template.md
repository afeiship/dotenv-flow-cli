# dotenv-flow-cli
> Cli templates for javascript.

## template
```sh
# clone the template
git clone git@github.com:aric-tpls/dotenv-flow-cli.git
mvg

# or use tiged
tiged git@github.com:aric-tpls/dotenv-flow-cli.git --mode=git --disable-cache

# replace the template
rifc
```
